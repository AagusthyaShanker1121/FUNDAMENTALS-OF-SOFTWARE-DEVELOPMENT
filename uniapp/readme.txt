# Enrolment Pro Max

Enrolment Pro Max is a Python-based application designed to help students manage their course enrolments. It provides features such as adding, dropping, and viewing enrolled courses.

## Features
- Add courses to your enrolment list.
- Drop courses from your enrolment list.
- View all currently enrolled courses.

## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/samwp95/FOSD.git
    ```
2. Navigate to the project directory:
    ```bash
    cd /Users/sam/Projects/FOSD
    ```
3. Ensure you have Python 3 installed on your system.

## Usage

To run the application, execute the following command from the root directory (`FOSD`):
    ```bash
    cd uniapp
    python main.py
    ```
